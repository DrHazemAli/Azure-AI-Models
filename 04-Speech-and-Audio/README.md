## üéµ Audio Requirements

For the best results with Azure Speech Service:
- **Audio Formats:** WAV, MP3, OGG, FLAC
- **Sample Rate:** 8kHz, 16kHz, 32kHz, 48kHz
- **Channels:** Mono or Stereo
- **Bit Depth:** 16-bit recommended
- **File Size:** Up to 1GB for batch processing

## üåç Supported Languages

Azure Speech Service supports:
- **Speech Recognition:** 100+ languages
- **Speech Synthesis:** 400+ voices in 140+ languages
- **Speech Translation:** 100+ languages
- **Custom Speech:** 100+ languages
